Modern Landing Page - Sənədləşdirmə

1. Giriş

Bu sənədləşmə "Modern Landing Page" şablonunu necə istifadə edəcəyinizi izah edir.

2. Quraşdırma

Şablonu quraşdırmaq üçün aşağıdakı addımları izləyin:

  1. Faylları endirin və `.zip` arxivdən çıxarın.
  2. `index.html` faylını açın.
  3. Şablonu öz serverinizə və ya yerli kompüterinizdə (`Live Server`, `XAMPP`, `MAMP` və s.) işlədin.

3. Fayl Strukturu

    / şablon-qovluğu
    ├── index.html
    ├── styles.css
    ├── script.js
    ├── img/
    ├── documentation.html

4. Şablonun Xüsusiyyətləri

  • Responsiv dizayn
  • Modern və minimalist stil
  • SEO optimizasiyası
  • CSS və JavaScript animasiyaları

5. Rəngləri və Fontları Dəyişdirmək

Şablonun rənglərini və fontlarını `styles.css` faylında dəyişə bilərsiniz:

    body {
        background-color: #f4f4f4;
        font-family: Arial, sans-serif;
    }
